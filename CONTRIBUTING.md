# Contributing Guidelines

## Issues

If you encounter an issue with this app, you are welcome to [raise an issue](https://github.com/IpshitaC/Chat-Room/issues/new).
Before that, please search for similar issues. It's possible somebody has encountered this issue already.

## Pull Requests

If you want to contribute to the repository, here's a quick guide:

1. [Fork the repo](https://github.com/IpshitaC/Chat-Room.git)
2. Develop your code changes.
3. Commit your changes
4. Push to your fork and [submit a pull request](https://github.com/IpshitaC/Chat-Room/pulls)
